# Privacy Policy for LexiScope

LexiScope is a Chrome extension designed to help users understand and learn from English content on webpages.

## Data Collection

LexiScope does not collect or store personal data on external servers.

The extension may process the following data:

- Selected text on webpages (when the user actively uses the extension)
- OCR-captured text (when the user triggers OCR)
- User settings such as API keys and preferences (stored locally in the browser)

## Data Usage

- Text selected or captured by the user may be sent to third-party AI providers (such as OpenAI, Claude, or others configured by the user) for analysis.
- This data is used solely to generate explanations, translations, or learning insights requested by the user.

## Data Storage

- All user-generated data (cards, tags, study progress) is stored locally in the user's browser using Chrome extension storage.
- LexiScope does not store user data on external servers.

## Third-Party Services

LexiScope allows users to configure their own API providers.

Any data sent to these providers is subject to their respective privacy policies.

## Data Sharing

LexiScope does not sell, share, or transfer user data to third parties.

## User Control

Users have full control over their data, including:

- Adding or removing API keys
- Deleting saved cards and data
- Uninstalling the extension at any time

## Changes

This privacy policy may be updated as the extension evolves.
